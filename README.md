# 🌐 Language Translator Web App  

This project is a **Language Translator Web Application** developed as part of an Artificial Intelligence Internship.  

The application allows users to enter text and translate it into multiple languages using **Python, Flask, and Deep Translator**.

---

## 🚀 Features  
- 🌍 Translate text into multiple languages  
- 🔎 Automatic language detection  
- 🤖 AI-powered translation using Deep Translator  
- 📋 Copy translated text to clipboard  
- 🧹 Clear text option  
- 🎨 Clean and simple user interface  

---

## 🛠 Technologies Used  
- Python  
- Flask  
- Deep Translator  
- HTML  
- CSS

## 📸 Project Demo  
The user can:

- Select target language
